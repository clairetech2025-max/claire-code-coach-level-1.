# CLAIRE Code Coach: Level 1

**Python Survival Kit for AI Coders**

CLAIRE Code Coach is a terminal-native learn-to-code package for people using AI-generated Python.

It teaches beginners how to run scripts, understand indentation, read error messages, and repair broken pasted code from tools like ChatGPT, Gemini, Claude, GitHub, Stack Overflow, and coding assistants.

The included **PyGyro** diagnostic engine detects structural problems such as:

- indentation drift
- tab/space collisions
- malformed blocks
- missing block bodies
- broken nesting
- markdown/code fence contamination
- AI paste corruption

This is not just a linter.

It is a beginner-friendly structural repair tutor for the AI coding era.
